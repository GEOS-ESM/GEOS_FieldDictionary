### Added

- `docs/schema-rfc-v0.2.0.md`: Schema RFC proposing v0.2.0 of the GEOS field dictionary YAML format, including new required fields (`long_name`, `conserved`, `verification_status`), renamed fields (`short_names` → `aliases`, `units` → `canonical_units`), new optional fields (`physical_dimension`, `provenance`), a top-level `field_dictionary` wrapper with version and metadata, and a migration strategy for the existing ~2,360 entries.
